# Gruha Alankara – Pre-Requisites Document

## Project Overview

Gruha Alankara is an AI and AR-powered Interior Design Platform built using Flask, Python, and local database architecture. The system integrates computer vision, NLP, agent-based automation, and AR visualization to provide personalized interior design recommendations without relying on cloud infrastructure.

This document outlines the essential tools, software, and knowledge required before starting development.

---

# 1. Technical Knowledge Requirements

## 1.1 Programming Fundamentals
- Strong understanding of Python
- Object-Oriented Programming (OOP)
- Functions and modular coding
- Exception handling
- Virtual environments (venv)
- JSON handling
- Basic REST API concepts

---

## 1.2 Web Development Basics
- Client–Server architecture understanding
- HTTP methods (GET, POST, PUT, DELETE)
- Basic HTML & CSS knowledge
- Flask framework basics:
  - Routing
  - Request & response handling
  - Template rendering (Jinja2)

---

## 1.3 Database Knowledge
- SQL fundamentals (CRUD operations)
- Table relationships (One-to-Many, Many-to-Many)
- Primary and Foreign Keys
- SQLite database usage
- SQLAlchemy ORM integration

---

## 1.4 AI / ML Concepts
- Basic understanding of:
  - Machine Learning inference (model usage, not training)
  - Transformers architecture (high-level understanding)
  - NLP fundamentals
  - Computer Vision basics
- Familiarity with:
  - Hugging Face Transformers
  - IBM Granite models
  - LangChain framework

---

## 1.5 API Integration
- Making API calls in Python
- Handling API keys securely
- Parsing JSON responses
- Integration of:
  - Google Text-to-Speech API
  - WebRTC API (camera access)

---

## 1.6 AR and 3D Visualization Basics
- Basic understanding of:
  - WebRTC browser camera integration
  - AR overlay concepts
  - 3D object placement logic
- Optional:
  - Three.js or similar 3D rendering tools

---

# 2. Software & Tools Required

## Development Environment
- Python 3.9 or above
- VS Code or PyCharm
- Git & GitHub
- pip package manager

## Python Libraries
- Flask
- SQLAlchemy
- SQLite3
- Transformers
- LangChain
- OpenCV
- Pillow (PIL)

# 3. System Requirements

- Minimum 8GB RAM (recommended for AI inference)
- Stable internet connection (for model downloads)
- Modern browser (Chrome or Edge for WebRTC support)

# 4. Conceptual Understanding

Before development begins, the developer should understand:

- Layered system architecture
- RESTful API design principles
- Modular backend structure
- Agent-based automation using LangChain
- Local deployment without cloud infrastructure
